# Actitud Fest Website

SPA website for the Actitud Fest music festival.

## Tech stack

- React
- Vite
- TailwindCSS v4
- React Router

## Pages

- Home
- Lineup
- Info
- About
- History

## Home sections

- Hero (video background)
- Lineup preview
- Days (Day 1 / Day 2)
- Info festival
- Stay in touch
- Supporters
- Footer

## Languages

- Catalan
- Spanish
- English

## Notes

- Simple layout
- Tailwind utilities only
- No backend required

## Project structure

src/
components/
pages/
assets/
images/
video/

## Coding rules

- One component per section
- Use TailwindCSS utilities only
- Keep components small and simple
